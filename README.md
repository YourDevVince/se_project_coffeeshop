# Triple Peaks Coffee Shop

This is a project I have done for the Software Engineering program at TripleTen. It was created using HTML and CSS, based on the design brief.

## Project features

- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM file structure
- A custom form
- CSS animation and transform

## Plan on improving the project

I plan on getting this site responsive for mobile devices soon.
